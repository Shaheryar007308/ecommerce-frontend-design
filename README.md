# 🛒 Ecommerce Frontend Design - DevelopersHub Internship

A pixel-perfect, fully responsive eCommerce web application built as part of the **DevelopersHub Corporation** internship program (March 2026). This project demonstrates the transition from a Figma design to a functional, interactive frontend experience.


## 📋 Project Overview
This project was completed over a 3-week sprint, focusing on core frontend technologies. The goal was to build a desktop-first design that scales beautifully across all devices, featuring a dynamic product flow from discovery to checkout.

### Key Features
- **Dynamic Product Rendering:** Products are loaded dynamically using JavaScript objects.
- **Persistent Shopping Cart:** Uses `localStorage` to keep items in the cart even after page refreshes.
- **Real-time Price Calculation:** Automatic updates for subtotal, tax, and discounts based on item quantity.
- **Responsive Design:** Optimized for Desktop, Tablet, and Mobile screens using CSS Grid and Flexbox.
- **Interactive Checkout:** A custom-built modal (pop-up) receipt that generates a summary upon successful "purchase."



## 🛠️ Tech Stack
- **HTML5:** Semantic structure for SEO and accessibility.
- **CSS3:** Custom styling with a focus on Grid/Flexbox layouts and Media Queries.
- **JavaScript (ES6+):** Logic for cart management, URL parameters, and DOM manipulation.
- **Font Awesome:** Professional iconography.
- **Figma:** Source for design specifications and assets.
